# Trumpet Simulator

A browser-based trumpet simulator that uses keyboard valves (J/K/L) and harmonic steps (W/S) to generate notes in real time. Includes a visual staff, live note display, and built‑in metronome.

## Controls
- J / K / L: valves 1 / 2 / 3
- W / S: harmonic up / down
- Shift: retrigger current note
- Metronome: start/stop + BPM slider

## Run
Open `index.html` in a browser.
